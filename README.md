## How to Use

1. Open `valentine.html` in your browser
2. Customize with URL parameters:
   ```
   valentine.html?name=YourName&valentineName=ValentineName
   ```

## Parameters
- `name`: Your name (who's asking)
- `valentineName`: Your valentine's name (who's being asked)

## Example
```
valentine.html?name=Nithish&valentineName=Roshin
```
